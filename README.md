# NexusAI: Neural Social Intelligence Platform

NexusAI is a cutting-edge social platform built for MLOps demonstration, integrating Machine Learning (ML), Deep Learning (DL), and Quantum Machine Learning (QML) into a single full-stack experience.

## 🚀 Key Features
- **Neural Sync (Auth)**: Secure login using Bcrypt hashing and cookie-based session management.
- **Satellite Feed**: Real-time social feed with AI insights.
- **DL Inference**: Automatic image classification using a pre-trained **MobileNetV2** model.
- **ML Vibe Analysis**: Sentiment analysis of posts using **TextBlob**.
- **Quantum Entanglement**: Friend recommendation engine powered by **PennyLane** QML circuits.
- **Parrot Green Theme**: A premium, high-contrast dark-mode aesthetic.

## 🛠️ Technology Stack
- **Backend**: FastAPI (Python)
- **Frontend**: Jinja2 Templates, Tailwind CSS, FontAwesome
- **AI/ML**: PyTorch (DL), TextBlob (ML), PennyLane (QML)
- **Security**: Bcrypt

## 📦 Setup & Deployment
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Launch the application:
   ```bash
   python app/main.py
   ```
3. Access the Nexus:
   `http://127.0.0.1:8000`

# 🪐 NexusAI: The Future of Neural Social Intelligence

**NexusAI** is a state-of-the-art full-stack MLOps demonstration platform. It blends social networking with high-performance Machine Learning, Deep Learning, and Quantum Computing to create a "Neural Social Intelligence" experience.

Developed as a comprehensive project for **MLOps CIE-3**, NexusAI showcases the complete lifecycle of AI models in a production-ready environment.

---

## 🚀 Intelligent Core Features

### 🧠 Deep Learning: MobileNetV2 Computer Vision
- **Automated Tagging**: Every image uploaded to the Nexus is analyzed in real-time using a pre-trained **MobileNetV2** architecture.
- **Contextual Intelligence**: The system recognizes complex objects and maps them to technical classifications (e.g., detecting hardware as "Quantum Hardware").
- **Confidence Scoring**: Each inference is accompanied by a probability metric, ensuring transparency in AI decision-making.

### 🎭 Machine Learning: Vibe Analysis
- **Sentiment Engine**: Powered by **TextBlob**, the system performs "Vibe Analysis" on every post.
- **Technical Boost**: Custom logic prioritizes technical discourse, marking high-quality engineering posts as "Progressive."

### ⚛️ Quantum Intelligence: PennyLane Entanglement
- **Friend Recommendations**: Utilizing **PennyLane** Quantum circuits, NexusAI calculates "Entanglement Scores" between users.
- **QML Circuits**: Uses qubit rotation and CNOT gates to determine compatibility in a high-dimensional feature space.

---

## 🛠️ MLOps Intelligence Loop

NexusAI isn't just a static app; it's a living model pipeline. The **MLOps Dashboard** provides:
- **Real-time Telemetry**: Monitor latency (ms), confidence levels (%), and total inference counts.
- **Active Learning**: Correct mislabeled AI predictions to improve the system.
- **Automated Retraining**: Initiate transfer learning on-the-fly to upgrade models from `v1.0.4` to `v1.1.0`.
- **System Rollback**: Instantly revert to stable model versions if performance drifts.
- **Data Augmentation**: Generate synthetic samples to balance datasets.
- **Audit Reports**: Export comprehensive system logs for compliance and debugging.

## 📦 Getting Started

### 1. Prerequisites
Ensure you have Python 3.9+ installed.

### 2. Installation
```bash
# Clone the repository
git clone https://github.com/Satishherakal/NexusAI-Full-Stack-MLOps.git

# Navigate to the project
cd NexusAI-Full-Stack-MLOps

# Install dependencies
pip install -r requirements.txt
```

### 3. Execution
```bash
python app/main.py
```
The Nexus will be available at: `http://127.0.0.1:8000`
